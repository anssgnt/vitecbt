MANUAL PUSH KE GITHUB - INSTRUKSI LENGKAP
==========================================

Karena ada permission issues dengan git CLI, gunakan salah satu metode berikut:

=== METODE 1: GITHUB DESKTOP (RECOMMENDED) ===

1. Download GitHub Desktop dari: https://desktop.github.com/
2. Install dan buka GitHub Desktop
3. Login dengan akun GitHub Anda
4. Click "File" -> "Clone Repository"
5. Paste: https://github.com/anssgnt/vitecbt.git
6. Click "Clone"
7. Buka folder C:\laragon\www\examkita di File Explorer
8. Copy semua file baru ke folder yang di-clone:
   - src/math-handler.js
   - src/arabic-handler.js
   - src/excel-importer.js
   - style-math-arab.min.css
   - supabase/functions/import_soal/index.ts
   - supabase/migrations/add_math_arab_excel.sql
   - Semua .md files
9. Update 3 HTML files:
   - index.html
   - exam.html
   - soal-editor.html
10. Kembali ke GitHub Desktop
11. Semua changes akan muncul di tab "Changes"
12. Di bagian bawah, masukkan commit message:
    "feat: Add Phase 2 & 3 - Matematika, Arab, Excel Import"
13. Click "Commit to main"
14. Click "Push origin"
15. Done!

=== METODE 2: VS CODE ===

1. Open VS Code
2. File -> Open Folder -> C:\laragon\www\examkita
3. Go to Source Control (Ctrl+Shift+G)
4. Click "+" icon untuk stage semua changes
5. Di message box, ketik:
   "feat: Add Phase 2 & 3 - Matematika, Arab, Excel Import"
6. Click checkmark untuk commit
7. Click "..." menu -> "Push"
8. Done!

=== METODE 3: GIT BASH ===

1. Download Git Bash dari: https://git-scm.com/download/win
2. Install Git Bash
3. Buka Git Bash
4. Ketik commands:
   cd C:/laragon/www/examkita
   git add .
   git commit -m "feat: Add Phase 2 & 3 - Matematika, Arab, Excel Import"
   git push origin main
5. Done!

=== METODE 4: TORTOISEGIT ===

1. Download TortoiseGit dari: https://tortoisegit.org/
2. Install TortoiseGit
3. Right-click folder C:\laragon\www\examkita
4. Select "Git Commit..."
5. Select all files
6. Enter commit message
7. Click "Commit"
8. Right-click folder lagi
9. Select "Git Push"
10. Done!
